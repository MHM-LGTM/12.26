# 快速测试 - 广场阶段一

## 🚀 一分钟快速测试

### 1. 刷新页面
```
按 F5 或 Ctrl+R
```

### 2. 查看广场面板
- [ ] 页面底部出现"动画广场"面板 ✅

### 3. 测试上传到广场
1. 鼠标悬浮"我的动画"中的任意卡片
2. 右上角出现"⋯"按钮
3. 点击"⋯" → 弹出菜单
4. 点击"📤 上传到广场"
5. 提示"已上传到动画广场"
6. 查看底部广场面板 → 应该出现该动画

**验收：**
- [ ] ✅ 菜单按钮显示
- [ ] ✅ 菜单弹出正确
- [ ] ✅ 上传成功
- [ ] ✅ 广场中能看到

### 4. 测试删除功能
1. 点击卡片"⋯" → "🗑️ 删除"
2. 确认删除
3. 动画消失

**验收：**
- [ ] ✅ 删除成功
- [ ] ✅ 列表刷新

### 5. 测试保存时询问
1. 运行新模拟
2. 点击"下载动画"
3. 填写名称，点击"保存"
4. 弹出询问弹窗："是否上传到广场？"
5. 选择"显示用户名"或"匿名"
6. 可选：勾选"不再提醒"
7. 点击"上传到广场"或"暂不了"

**验收：**
- [ ] ✅ 询问弹窗显示
- [ ] ✅ 选项正常工作
- [ ] ✅ 上传成功显示在广场
- [ ] ✅ "不再提醒"功能有效
